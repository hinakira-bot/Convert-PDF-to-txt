# PDF to Text Converter

複数のPDFファイルを1つのテキストファイルに変換するブラウザベースツール。

## 機能

- 📎 複数のPDFファイルをドラッグ&ドロップ / ファイル選択で追加
- 📄 PDF.js（Mozilla製）でブラウザ内テキスト抽出
- 🔄 縦書きテキストの横書き自動変換
- 📝 成文処理（不要な改行の整理・文の結合）
- 🇯🇵 日本語CMapフォント対応（文字化け防止）
- 💾 1つの `.txt` ファイルとしてダウンロード
- 📊 処理統計（ファイル数、ページ数、文字数、出力サイズ）

## 使い方

1. `index.html` をブラウザで開く（またはVercelデプロイURL）
2. PDFファイルをドラッグ&ドロップ（または「ファイルを選択」）
3. 「⚡ テキストに変換」をクリック
4. プレビューで確認後「💾 ダウンロード」

## 技術

- **フロントエンドのみ**: HTML5 / CSS3 / Vanilla JavaScript
- **PDF解析**: [PDF.js v3](https://mozilla.github.io/pdf.js/)（CDN）
- **サーバー不要**: すべてブラウザ内で処理、PDFデータは外部送信されません

## 対応PDF

| 種類 | 対応 |
|---|---|
| テキストベースPDF | ✅ |
| 縦書きPDF | ✅（横書き変換） |
| スキャン画像PDF | ⚠️ OCR別途必要 |
